# Kidney & Tumor Segmentation Post-processing

신장(kidney) + 종양(tumor) + 물혹(cyst) 세그멘테이션 라벨의 후처리 도구.

> English version: [README_EN.md](README_EN.md)

## 폴더 구조

```
segmentation/
├── S003/
│   ├── S003_Segmentation_A.nii.gz    # 세그멘테이션 (A phase)
│   ├── S003_image_A.nii.gz           # CT 이미지 (A phase)
│   └── backup_original/              # 원본 백업 (자동 생성)
├── S004/
│   ├── S004_segmentation_A.nii.gz
│   ├── S004_image_A.nii.gz
│   └── ...
└── segtools.py
```

### 라벨 정의

| 라벨 | 이름 | 설명 |
|------|------|------|
| `0` | 배경 | 장기 외부 |
| `1` | 신장 (kidney) | 신장 실질 |
| `2` | 종양 (tumor) | 신장 내 종양 |
| `3` | 물혹 (cyst) | 신장 내 물혹 |

---

## 실행 방법

```bash
python segtools.py <케이스 폴더>
```

```bash
# 예시
python segtools.py S004
```

### 사용 흐름

```
1. Phase 선택 (A / D / P / all=전체)
2. 기능 번호 선택
3. 하위 선택지 및 파라미터 입력 (엔터로 기본값 사용)
4. 실행 → 자동 백업 → 자동 저장
5. 결과 확인 후 enter(계속) 또는 r(롤백)
6. 다음 기능 진행 또는 q로 종료
```

- 모든 입력 단계에서 `q`를 입력하면 현재 기능을 취소합니다.
- 모든 입력 단계에서 `r`을 입력하면 직전 상태로 롤백합니다.
- 최초 실행 시 원본 파일이 `backup_original/` 폴더에 자동 백업됩니다.

## 기능 상세 설명

### 기능 1: 라벨 상태 분석

현재 세그멘테이션의 상태를 출력합니다. 데이터를 변경하지 않습니다.

**출력 내용:**
- 라벨별 voxel 수
- CT 이미지가 있는 경우 라벨별 intensity 평균 ± 표준편차 (HU)
- 신장/종양 각각의 connected component 개수
- 상위 5개 component의 크기
- 표면 비율 (surface ratio) — 값이 높을수록 경계가 불규칙

**추가 입력 없음** — 바로 실행됩니다.

---

### 기능 2: 고립 복셀 제거

신장이나 종양의 connected component 분석 후, 상위 N개만 유지하고 나머지 작은 고립 덩어리를 배경(0)으로 삭제합니다.

**선택지:**

| 입력 | 대상 | 유지 개수 |
|------|------|----------|
| `1` | 신장 (label 1) | 상위 2개 (좌/우 신장) |
| `2` | 종양 (label 2) | 상위 1개 |
| `3` | 둘 다 | 신장 2개 + 종양 1개 |

**추가 파라미터 없음** — 선택 후 바로 실행됩니다.

---

### 기능 3: 저강도 제거

CT intensity가 0 HU 이하인 신장/종양 복셀을 배경(0)으로 삭제합니다.
공기, 지방 등 장기 조직이 아닌 영역을 정리하는 데 사용합니다.

**필수 조건:** CT 이미지 파일이 있어야 실행 가능합니다.

**추가 입력 없음** — 바로 실행됩니다. threshold는 0으로 고정입니다.

---

### 기능 4: 고강도 제거

CT intensity가 threshold 이상인 신장/종양 복셀을 배경(0)으로 삭제합니다.
석회화, 조영제가 과도하게 들어간 혈관 등을 제거하는 데 사용합니다.

**필수 조건:** CT 이미지 파일이 있어야 실행 가능합니다.

**파라미터:**

| 파라미터 | 설명 | 기본값 |
|----------|------|--------|
| Threshold | 이 값 이상인 복셀을 삭제 (HU) | 400 |

---

### 기능 5: Smoothing

morphology 연산(closing, opening)과 Gaussian smoothing을 조합하여 경계를 부드럽게 다듬습니다.

**1단계 — 대상 선택:**

| 입력 | 대상 | 동작 |
|------|------|------|
| `1` | 종양 (label 2) | 종양 경계만 smoothing. 가장 큰 component만 유지. 결과는 신장+종양 영역 내로 제한 |
| `2` | 물혹 (label 3) | 물혹 경계 smoothing. 신장+물혹 영역 내로 제한. 내부 구멍도 자동 채움 |
| `3` | 장기 전체 외곽 | 신장+종양+물혹을 합쳐서 외곽 smoothing. 종양/물혹 라벨은 보존 |

**2단계 — 파라미터 입력:**

| 파라미터 | 설명 | 종양 기본값 | 물혹 기본값 | 장기 전체 기본값 |
|----------|------|-----------|-----------|----------------|
| Gaussian sigma (mm) | smoothing 강도. 클수록 부드러움 | 1.0 | 1.0 | 1.0 |
| Closing 반복 | 구멍/빈 부분 채우기 강도 | 3 | 2 | 3 |
| Opening 반복 | 돌출부 제거 강도 | 2 | 1 | 2 |
| 유지할 component 수 | (장기 전체만) 상위 N개 component 유지 | - | - | 2 |

**파라미터 조정 팁:**
- 경계가 거칠면 → sigma를 높이기 (1.5~2.0)
- 구멍이 많으면 → closing 반복 높이기 (4~5)
- 움푹 패이면 → opening 반복 줄이기 (0~1)
- 볼륨이 과도하게 줄면 → opening 반복 줄이기

---

### 기능 6: 경계 확장

선택한 라벨의 경계를 1복셀씩 반복 dilation하면서, intensity 조건을 만족하는 복셀만 흡수합니다.

**1단계 — 대상 선택:**

| 입력 | 대상 | 확장 가능 영역 |
|------|------|--------------|
| `1` | 신장 (label 1) | 배경(0)만 |
| `2` | 종양 (label 2) | 배경(0) + 신장(1) |
| `3` | 물혹 (label 3) | 배경(0) + 신장(1) |

대상 선택 후 현재 라벨의 intensity 통계(평균 ± 표준편차 HU)가 자동으로 출력됩니다.

**2단계 — Intensity 조건 선택:**

| 입력 | 조건 | 용도 |
|------|------|------|
| `1` | 하한만 (≥ threshold) | 특정 HU 이상 복셀만 흡수. 신장 확장에 적합 |
| `2` | 양방향 범위 (평균 ± tolerance) | 라벨과 비슷한 HU인 복셀만 흡수. 물혹/종양 확장에 적합 |

**3단계 — 파라미터 입력:**

하한만 모드:

| 파라미터 | 설명 | 기본값 |
|----------|------|--------|
| 최소 intensity HU | 이 값 이상인 복셀만 흡수 | 120 |

양방향 범위 모드:

| 파라미터 | 설명 | 기본값 |
|----------|------|--------|
| 허용 HU 범위 (평균 ± X) | 평균에서 이 범위 내 복셀만 흡수 | max(표준편차×2, 15) |

공통:

| 파라미터 | 설명 | 기본값 |
|----------|------|--------|
| 확장 횟수 | 매 회 경계에서 1복셀씩 확장 반복 | 5 |

---

### 기능 7: 경계 축소

선택한 라벨의 표면에서 HU 범위 밖의 복셀을 바깥부터 1겹씩 반복 깎아냅니다.

**1단계 — 대상 선택:**

| 입력 | 대상 | 깎인 복셀 처리 |
|------|------|--------------|
| `1` | 신장 — 장기 전체 외곽 (신장+종양+물혹) | 모두 배경(0)으로 |
| `2` | 종양 (label 2) | 배경 인접 → 배경(0), 나머지 → 신장(1) |
| `3` | 물혹 (label 3) | 배경 인접 → 배경(0), 나머지 → 신장(1) |

대상 선택 후 해당 라벨의 intensity 통계(평균 ± 표준편차 HU)가 자동으로 출력됩니다.

**2단계 — 파라미터 입력:**

| 파라미터 | 설명 | 기본값 |
|----------|------|--------|
| 허용 HU 범위 (평균 ± X) | 이 범위 밖 표면 복셀을 깎아냄 | max(표준편차×2, 15) |
| 최대 반복 | 표면 1겹씩 반복 깎기 | 1 |

---

### 기능 8: 경계 계단 메꿈

장기 외곽(신장+종양)에서 슬라이스 사이 계단 형태의 울퉁불퉁함을 26-connectivity closing으로 메꿉니다.
채워진 복셀은 신장(1)으로 라벨링되며, 종양/물혹은 보호됩니다.

**필수 조건:** CT 이미지가 있으면 HU 필터가 적용됩니다.

**파라미터:**

| 파라미터 | 설명 | 기본값 |
|----------|------|--------|
| Closing 반복 | 채우기 강도. 1이면 대각 1복셀 틈새만 채움 | 1 |
| 최소 intensity HU | 이 값 이상인 복셀만 채움 | 120 |

---

### 기능 9: 돌출부 제거

신장(label 1) 경계에서 얇게 돌출된 부분을 morphology opening으로 제거합니다.
종양에 인접한 신장 복셀은 보호됩니다.

**파라미터:**

| 파라미터 | 설명 | 기본값 |
|----------|------|--------|
| Opening 반복 | 제거 강도. 클수록 더 두꺼운 돌출부도 제거 | 1 |

---

### 기능 10: 내부 구멍 채우기

라벨 내부의 빈 공간(구멍)을 `binary_fill_holes`로 채웁니다.

**선택지:**

| 입력 | 대상 | 동작 |
|------|------|------|
| `1` | 신장 (label 1) | 신장 내부 구멍을 신장으로 채움 |
| `2` | 종양 (label 2) | 종양 내부 구멍을 종양으로 채움 |
| `3` | 신장+종양 장기 전체 | 장기 전체를 합쳐서 채움. 채워진 부분은 신장(1)으로. 물혹 보호 |
| `4` | 물혹 (label 3) | 물혹 내부 구멍을 물혹으로 채움 |

**추가 파라미터 없음** — 선택 후 바로 실행됩니다.

---

### 기능 11: 고립 신장 → 종양 재라벨링

신장 connected component 중 상위 2개(좌/우 신장 본체)를 제외한 고립 component 중에서, 종양과 인접한 것만 종양(2)으로 재라벨링합니다.

종양 세그멘테이션이 신장으로 잘못 라벨링된 경우를 교정하는 데 사용합니다.

**추가 입력 없음** — 바로 실행됩니다.

---

### 기능 12: 볼록 껍질 기반 라벨링

이미지 뷰어에서 미리 칠해둔 시드(seed) 복셀을 기반으로 3D Convex Hull(볼록 껍질)을 생성하고, 그 내부를 채워 라벨링합니다.

**사전 준비:** 이미지 뷰어(ITK-SNAP 등)에서 종양은 label 2, 물혹은 label 3으로 여러 평면(axial/sagittal/coronal)에 걸쳐 시드를 칠해야 합니다.

**1단계 — 대상 선택:**

| 입력 | 대상 | 시드 라벨 | 허용 영역 | 보호 라벨 |
|------|------|----------|----------|----------|
| `1` | 종양 (label 2) | label 2 | 신장 + 종양 | 물혹(3) 침범 방지 |
| `2` | 물혹 (label 3) | label 3 | 신장 + 물혹 | 종양(2) 침범 방지 |

**추가 파라미터 없음** — 선택 후 자동으로 Convex Hull 계산 및 채움이 실행됩니다.

**주의:** 시드가 한 평면에만 있으면 Convex Hull이 납작해져서 실패할 수 있습니다. 반드시 여러 평면에 걸쳐 시드를 칠해주세요.

---

### 기능 13: 세그멘테이션 합치기

외부 NIfTI 파일에서 신장(label 1) 라벨을 가져와, 현재 세그멘테이션의 신장을 교체합니다. 현재 파일의 종양(2)과 물혹(3)은 유지됩니다.

서로 다른 모델이나 방법으로 생성한 신장/종양 세그멘테이션을 합칠 때 사용합니다.

**입력:**

| 파라미터 | 설명 |
|----------|------|
| 파일 경로 | 신장 라벨을 가져올 NIfTI 파일의 경로 |

**주의:** 외부 파일과 현재 파일의 shape가 일치해야 합니다.

---

### 기능 14: Phase 비교 분석

모든 phase(A, D, P)의 세그멘테이션을 동시에 로드하여 비교 분석합니다.
데이터를 변경하지 않습니다. phase 선택과 무관하게 실행됩니다.

**출력 내용:**

1. **Phase별 라벨 크기 비교** — 신장/종양/물혹/장기전체 각각의 voxel 수를 phase별로 나란히 출력. phase간 최대 차이(%)도 표시.

2. **Phase 쌍별 Dice 계수** — A↔D, A↔P, D↔P 등 모든 쌍에 대해 각 라벨별 Dice 계수(0~1) 출력. 1에 가까울수록 일치.

3. **불일치 영역 슬라이스 분석** — 각 phase 쌍, 각 라벨별로 차이가 있는 슬라이스 범위와 가장 차이가 큰 상위 5개 슬라이스 출력.

**추가 입력 없음** — 바로 실행됩니다.

---

### 롤백 (r)

직전 상태로 되돌립니다. 여러 단계의 롤백이 가능합니다.
기능 실행 후에도 바로 `r`을 입력하여 롤백할 수 있습니다.
